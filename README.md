# Pacman-Robotics
This ROS2 project features a physical robot playing Pac-Man on a projected field. A camera analyzes the game state, while `damt_game_msg_main` uses `int16` messages to save bandwidth. Key nodes include `damt_vision_main`, `damt_map_main`, `pacman_main`, `damt_drive_main`, and `damt_bringup_main` for full system integration.
